# TokenDock 1.1.0

## What's New

- Added Windows startup launch option in Settings.
- Added Windows taskbar band display for quick Codex/Claude usage checks.
- Added taskbar band display mode setting: remaining usage or used usage.
- Added live Windows light/dark theme handling for taskbar band text colors.
- Added per-provider taskbar indicators matching the mini widget colors.

## Improvements

- Taskbar band now shows Codex and Claude on separate compact rows.
- Taskbar band shows `5h` and `7d` values for each provider.
- Usage text color now reflects risk based on the selected display mode.
- Taskbar band uses a transparent background to better blend with Windows 11 taskbar styling.
